Запуск бредогенератора
Для того ,чтобы запустить программу, откройте консоль и напишите: py console.py 
-l текст для обучения
-s количество предложений: если указано одно число, то будет нужное число предложений, если два, то случайное число предложений в диапазоне
-n число n-gram

пример запуска
py console.py -l warandpeace -s 10
py console.py -l warandpeace -s 3 10
